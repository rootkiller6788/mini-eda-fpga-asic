# mini-noc-design - Network-on-Chip Design (C99)
Network-on-Chip design: topology construction, router microarchitecture, routing algorithms, flow control, and performance analysis.
## Features
- **Topology**: Mesh, Torus, Ring, Crossbar with diameter/bisection analysis
- **Router**: Multi-port router with VC buffers, credit-based flow control
- **Routing**: XY, Odd-Even, West-First, North-Last, Adaptive algorithms
- **Flow Control**: Credit-based, Wormhole, VCT mechanisms
- **Performance**: Packet injection, latency/throughput tracking, saturation analysis
## Build
`make && make test`
