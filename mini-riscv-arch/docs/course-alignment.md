# Course Alignment — mini-riscv-arch

## UC Berkeley CS152: Computer Architecture
- CSR register file implementation: mstatus/mcause/mepc/mtvec
- Privilege modes: M/S/U with trap delegation

## MIT 6.191: RISC-V Architecture
- Exception handling: 12 standard exception types
- Interrupt controller: software/timer/external IRQs
- Sv32 page table walk and TLB simulation

## CMU 18-447: Computer Architecture
- MMU design: address translation, TLB, page permissions
- PMP (Physical Memory Protection) checks

## RISC-V Privileged Specification v1.12
- Debug Module (DM) and Debug Transport Module (DTM)
- Abstract command register access
- Hardware breakpoint support
